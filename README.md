## Використані технології
HTML5:
Семантична розмітка (header, main, section, footer).
Адаптивні метатеги (viewport).
Вбудована валідація форм (атрибути required, type="email", pattern).

CSS:
Зовнішня стилізація через підключення CSS-файлів.
Брендова стилістика: змінні :root для кольорів та шрифтів.
Flexbox — для побудови гнучких макетів та адаптивних блоків.
Медіазапити (@media) — для адаптивного відображення на мобільних телефонах, планшетах і ПК.
Адаптивне бургер-меню для мобільних пристроїв.
Стилізація форм: поля вводу, кнопки, повідомлення про помилки, відображення статусу "успішно відправлено".

JavaScript:
Скрипт для бургер-меню (відкривання/закривання).
Універсальна обробка всіх форм і показ повідомлення про успішне відправлення.

# JavaScript
Робота з DOM: динамічне додавання елементів, зміна стилів карток рослин, автоматичне відображення поточної дати у футері.

Події та обробники:
Клік по кнопці "Показати більше" — розкриває прихований текст про рослини.
Клік по кнопці "Змінити тему" — перемикає між світлою та темною темою (dark/light) зі збереженням вибору у localStorage.
Підсвічується меню навігації при наведенні миші через JS-класи.
Змінюється розміру шрифту сторінки при натисканні клавіш ArrowUp/ArrowDown.

Робота з формами та валідація:
Забороняється стандартна відправка форм (event.preventDefault()).
Зчитуються дані з усіх полів.
Клієнтська валідація: перевіряється мінімальна довжина і правильність email.
Підсвічуються поля з помилками та відобрується текст помилки під полем.
Виводиться повідомлення "Форма успішно надіслана!" після успішної валідації.

LocalStorage:
Збереження вибору теми сайту (dark/light) між сеансами.
